# CrankIssues
Issue tracking for Crank remake in Godot.

Thanks for coming by to submit an issue with Crank! I do my best to make it bug/issue free, but stuff will always slip by.

To submit an issue or bug report, choose "Issues" form the top left of this page.

When submitting an issue, provide as much detail as posseble, such as:

- What's not working?
- What you expect to happen when it is working?
- What version of the game you're running.
- Any screenshots showing the issue happening.
- An export of your save file.
